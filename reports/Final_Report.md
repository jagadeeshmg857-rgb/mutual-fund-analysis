## Executive Summary

This project focuses on analyzing mutual fund performance, investor behavior, portfolio concentration, and SIP trends. The solution includes data ingestion, cleaning, exploratory data analysis, performance analytics, dashboard development, and advanced risk metrics such as VaR, CVaR, Sharpe Ratio, and HHI Concentration Index.

The project provides actionable insights for investors and fund managers through visual dashboards and analytics.

## Project Objective

- Analyze mutual fund performance.
- Track AUM growth and SIP inflows.
- Measure fund risk and returns.
- Identify investor behavior patterns.
- Build a recommendation engine.
- Create an interactive dashboard.
